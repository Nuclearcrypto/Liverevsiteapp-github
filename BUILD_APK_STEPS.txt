REV-N-RIP FULL SITE APP BUILD STEPS

1. Download and unzip this package.
2. Open Android Studio.
3. Click Open.
4. Select the unzipped project folder containing settings.gradle.
5. Let Gradle sync.
6. Build > Build Bundle(s) / APK(s) > Build APK(s).

The APK will be here:
app/build/outputs/apk/debug/app-debug.apk

This version uses the full website content inside:
app/src/main/assets/
